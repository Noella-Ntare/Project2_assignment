# Project2_Assignment

## Projects
- **Project 1** - Dynamic Student Record Engine
- **Project 2** - Adaptive Text Intelligence Tool  
- **Project 3** - Callback-Based Device Monitoring Simulator

## How to Compile and Run

### Project 1
gcc student_records.c -o student_records -lm
./student_records

### Project 2
gcc text_intelligence.c -o text_intelligence
./text_intelligence

### Project 3
gcc device_monitor.c -o device_monitor
./device_monitor

## Requirements
- GCC compiler
- Terminal that support C

## What each project demonstrates
- Structs, dynamic memory (malloc/realloc/free)
- Function pointers
- Pointer traversal
- String functions (strcmp, strcpy, strlen)
- Unions and callbacks
