# C Programming Projects

## Projects
- **Project 1** - Dynamic Student Record Engine
- **Project 2** - Adaptive Text Intelligence Tool  
- **Project 3** - Callback-Based Device Monitoring Simulator

## How to Compile and Run

### Project 1
gcc project1_student_records.c -o project1 -lm
./project1

### Project 2
gcc project2_text_intelligence.c -o project2
./project2

### Project 3
gcc project3_device_monitor.c -o project3
./project3

## Requirements
- GCC compiler
- Terminal that support C

## What each project demonstrates
- Structs, dynamic memory (malloc/realloc/free)
- Function pointers
- Pointer traversal
- String functions (strcmp, strcpy, strlen)
- Unions and callbacks
